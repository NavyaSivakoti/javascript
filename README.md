## Topics Covered

### Core JavaScript (Sessions 1-7)
- JavaScript fundamentals - how it runs, V8 engine architecture
- Variable declarations - `var`, `let`, `const`, scoping and hoisting
- Data types, type coercion, `typeof`
- Operators - arithmetic, template literals, equality (`==` vs `===`), increment/decrement
- Conditionals and switch case
- Loops and truthy/falsy values
- Arrays and common array methods

### Objects and Functions (Sessions 8-11)
- 2D arrays, object creation, nested objects, object references
- JSON serialization and deserialization
- Functions - types, parameters vs arguments, return values
- Heap and stack memory, call stack
- Default parameters, function expressions, `this` keyword

### Advanced Functions (Sessions 12-14)
- Optional chaining (`?.`), rest parameters (`...`)
- IIFE, anonymous functions, arrow functions
- Callback functions - utility pattern, inline callbacks
- `map`, `filter`, `reduce` - chaining array methods
- Call by reference vs call by value

### Asynchronous JavaScript (Sessions 15-17)
- Callback hell and why it's a problem
- Promises - pending, resolved, rejected states
- `.then()`, `.catch()`, `.finally()` chaining
- `Promise.all()`, `Promise.race()`, `Promise.any()`, `Promise.allSettled()`
- `async/await` - writing async code that reads like sync code

### Modern JavaScript and OOP (Sessions 18-19)
- Array and object destructuring
- Export and import - named, default, wildcard
- Classes and objects, constructors
- Inheritance, method overriding
- Encapsulation - private variables and methods (`#`)
- Static properties and methods
